# java-games-for-datastruct
----
## Installation Guide
  1. Proceed to [Releaeses](https://github.com/Lucrintoko/java-games-for-datastruct/releases) and install the `JavaGames.zip`
  2. Un-zip it to your preffered location
  3. Open `JavaGames` Folder and launch **JavaGames.exe**
  4. Enjoy the Game!
